# Sales_Data_Analysis

## Objective
The objective of this project is to analyze a sales dataset to identify monthly trends, best-selling products, and region-wise performance. The project focuses on data cleaning, aggregation, and visualization using Python libraries.

## Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The dataset contains sales records with columns such as:
- Order_Date
- Region
- Category
- Product
- Customer ID
- Quantity
- Sales

## Workflow
1. Import libraries
2. Load dataset
3. Data cleaning (missing & inconsistent values)
4. Feature engineering (extract month & year)
5. Exploratory Data Analysis (EDA)
6. Data visualization
7. Conclusion & insights
